Personalización de Terminal Linux

> Guía paso a paso para transformar tu entorno de trabajo.

Esta guía te enseñará a instalar y configurar Oh My Zsh y Powerlevel10k para tener una terminal potente, rápida y visualmente atractiva, ideal para operaciones de Ciberseguridad.

01 Instalación de Zsh

Primero, necesitamos asegurarnos de tener Zsh instalado en nuestro sistema. Abre tu terminal y ejecuta:

# Actualizar repositorios
sudo apt update && sudo apt upgrade -y

# Instalar Zsh
sudo apt install zsh -y
# También podemos visualizar la versión con 'zsh --version'

# Establecer Zsh como shell por defecto
chsh -s $(which zsh)

02 Instalar Oh My Zsh

Oh My Zsh es un framework para gestionar tu configuración de Zsh. Usa el siguiente comando curl para instalarlo:

sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Nota: Al terminar, cierra y vuelve a abrir tu terminal para ver los cambios.

03 Tema Powerlevel10k

Clona el repositorio de Powerlevel10k dentro de la carpeta de temas de Oh My Zsh.

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

Luego, edita tu archivo ~/.zshrc y cambia la variable ZSH_THEME:

nano ~/.zshrc

Busca la línea que dice ZSH_THEME="robbyrussell" y cámbiala por:

ZSH_THEME="powerlevel10k/powerlevel10k"

04 Configuración Final

Reinicia tu terminal. El asistente de configuración de P10k se iniciará automáticamente o puedes usar 'p10k configure'. Sigue los pasos en pantalla para elegir el estilo que más te guste (estilo "Rainbow" recomendado para hackers).

04.1 Instalar Fuentes Necesarias (Crucial)

Powerlevel10k requiere una fuente con soporte de glifos y símbolos especiales. La fuente recomendada es MesloLGS NF.


Descargar e Instalar las Fuentes:

# Importante: cada 'curl' va en una sola línea hasta terminar el '.ttf'
# tener en cuenta que '~/Descargas' y '~/Downloads' depende del idioma instalado
cd ~/Descargas && {
  curl -fsSL https://github.com/romkatv/powerlevel10k-media/raw/master/MesloLGS%20NF%20Regular.ttf -o MesloLGS\ NF\ Regular.ttf
  curl -fsSL https://github.com/romkatv/powerlevel10k-media/raw/master/MesloLGS%20NF%20Bold.ttf -o MesloLGS\ NF\ Bold.ttf
  curl -fsSL https://github.com/romkatv/powerlevel10k-media/raw/master/MesloLGS%20NF%20Italic.ttf -o MesloLGS\ NF\ Italic.ttf
  curl -fsSL https://github.com/romkatv/powerlevel10k-media/raw/master/MesloLGS%20NF%20Bold%20Italic.ttf -o MesloLGS\ NF\ Bold\ Italic.ttf
}

# Para instalar las fuentes en el sistema:
sudo mkdir -p /usr/share/fonts/truetype/meslo
sudo mv ~/Descargas/MesloLGS*.ttf /usr/share/fonts/truetype/meslo/
sudo fc-cache -fv

EXTRA PLUGINS

Ahora viene la parte divertida: añadir las funcionalidades de "sugerencias" y "resaltado de sintaxis".

# Zsh Autosuggestions
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

# Zsh Syntax Highlighting
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

# Activar los Plugins
nano ~/.zshrc

Dentro del 'zshrc' buscaremos la sección 'plugins=(git ...)'

plugins=(
  git
  zsh-autosuggestions
  zsh-syntax-highlighting
)

Guarda los cambios (Ctrl + O, Ctrl + X) y recarga la configuración:

source ~/.zshrc
