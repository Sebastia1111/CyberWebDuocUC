Personalización de Terminal Linux

> Guía paso a paso para transformar tu entorno de trabajo.

Esta guía te enseñará a instalar y configurar Oh My Zsh y Powerlevel10k para tener una terminal potente, rápida y visualmente atractiva, ideal para operaciones de Ciberseguridad.

01 Instalación de Zsh

Primero, necesitamos asegurarnos de tener Zsh instalado en nuestro sistema. Abre tu terminal y ejecuta:

# Actualizar repositorios
sudo apt update && sudo apt upgrade -y

# Instalar Zsh
sudo apt install zsh -y

# Establecer Zsh como shell por defecto
chsh -s $(which zsh)

02 Instalar Oh My Zsh

Oh My Zsh es un framework para gestionar tu configuración de Zsh. Usa el siguiente comando curl para instalarlo:

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Nota: Al terminar, cierra y vuelve a abrir tu terminal para ver los cambios.

03 Tema Powerlevel10k

Clona el repositorio de Powerlevel10k dentro de la carpeta de temas de Oh My Zsh.

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

Luego, edita tu archivo ~/.zshrc y cambia la variable ZSH_THEME:

nano ~/.zshrc

Busca la línea que dice ZSH_THEME="robbyrussell" y cámbiala por:

ZSH_THEME="powerlevel10k/powerlevel10k"

04 Configuración Final

Reinicia tu terminal. El asistente de configuración de P10k se iniciará automáticamente. Sigue los pasos en pantalla para elegir el estilo que más te guste (estilo "Rainbow" recomendado para hackers).
